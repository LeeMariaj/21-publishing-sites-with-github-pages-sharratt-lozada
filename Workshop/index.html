TheWeatherDataSite



The purpose of this website is to explain how different variables can affect the behaviour of the weather in different cities around the world.

The analysis was made by retrieving, manipulating and analyzing an open-to-public API called "The Weather API".

Firstly, an account was created in such website providing a unique API Key used to get access to the information and make it possible to manipulate it for the further analysis.

The tool used to make the data wrangling was a Jupyter Notebook along with the Pandas and Matplotlib libraries using Python as the coding language.

The idea was to plan a ficticious holiday trip in such a manner that parameters where needed to enter to get the best fits for the perfect holiday trip.

Some of the parameters entered into the code were:


  • Minimum temperature
  • Maximum temperature
  • Latitude
  • Longitude
  • Precipitation
  • Humidity

The end results display 4 different correlation analyses each one having a single, unique parameter entered: latitude, humidity, cloudiness and windspeed.

Google Places was used for the second part of the assignment in such way that once all the locations were found that fitted the base criteria, a list of hotels would display in those resulting cities with hotel data such as the Hotel Name and the Hotel Rating.

Finally, a world map would be displayed with all of those cities pointed out with markers that once the user hover on those markers, a message would appear showing the City Name and Hotel information.
